# LocaleSwitcher

## Italiano

Questo file eseguibile consente di modificare rapidamente il locale di sistema di Windows tra le seguenti opzioni:

1) Italiano		– it-IT (locale principale del sistema)
2) Giapponese	– ja-JP
3) Inglese		– en-US

Descrizione:
Si tratta di uno script PowerShell progettato per cambiare il locale di Windows senza la necessità di installare ulteriori pacchetti linguistici.
Grazie a una interfaccia grafica intuitiva (GUI), il cambio del locale avviene in maniera rapida e automatica al termine della sessione, senza richiedere un riavvio del sistema.

Nota: il cambio automatico non viene applicato in caso di crash o altri errori imprevisti.

Requisiti:
Avvio dello script con permessi di amministratore per garantire il corretto funzionamento.

## English

This executable allows you to quickly switch the system locale in Windows among the following options:

1) Italian	– it-IT (system default locale)
2) Japanese	– ja-JP
3) English	– en-US

Description:
This is a PowerShell script designed to change the Windows locale without the need to install additional language packs.
Through an intuitive graphical user interface (GUI), the locale change occurs quickly and automatically upon closing the application, without requiring a system restart.

Note: automatic switching will not occur in the event of a crash or other unexpected errors.

Requirements:
Run the script with administrator privileges to ensure proper functionality.
